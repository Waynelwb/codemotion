# CodeMotion 项目规划

## 项目目标
Flutter 跨平台 C++ 可视化学习应用

## 已完成
- ✅ Landing Page 完整 UI (单文件 main.dart)

## 待完成

### 1. 美工设计组任务
- 设计系统：颜色、字体、间距规范
- 组件设计：课程卡片、代码编辑器、可视化动画组件
- 算法可视化设计：排序/查找等算法的动态演示 UI
- 交互动效：页面过渡、hover 效果、微交互
- 输出：`lib/design/` 目录下的设计组件

### 2. 内容撰写组任务
- C++ 基础语法课程（变量、数据类型、运算符、控制流）
- 面向对象编程课程（类、对象、继承、多态）
- STL 容器课程（vector、map、set、algorithm）
- 排序算法：冒泡、选择、插入、快排、归并
- 查找算法：顺序、二分
- 输出：`lib/content/` 目录下的课程数据

### 3. 主理人（我）整合任务
- 将设计组件与内容数据整合
- 创建页面路由和导航
- 实现核心页面（课程列表、课程详情、可视化演示）
- 集成算法可视化动画逻辑

## 技术栈
- Flutter + Dart
- google_fonts
- 状态管理：Provider（轻量级）
- 页面路由：go_router
