# CodeMotion 🚀

从零开始的 C++ 学习网站，带你一步步掌握 C++ 语法到高级特性，并通过动态可视化展示算法原理。

## 特点

- 📚 循序渐进的内容设计
- 🎨 算法可视化演示
- 💻 交互式代码示例
- 🌐 跨平台支持（Web / iOS / Android / Desktop）

## 技术栈

- **Flutter** - 跨平台 UI 框架
- **Dart** - 编程语言

## 本地运行

```bash
flutter pub get
flutter run
```

## 学习路线

- 基础语法 → 面向对象 → STL → 算法进阶
